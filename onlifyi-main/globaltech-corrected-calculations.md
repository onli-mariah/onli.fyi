# GlobalTech Corrected Calculations (Asset-Based Pricing)

## Key Assumptions
- **Asset Denomination**: $500 per OnliCredit token
- **Average Transaction**: $250,000 (= 500 tokens)
- **Onli Pricing**: $0.01 per token (one-time issuance cost)

## Pilot Program: $10 Billion

### Number of Tokens
```
$10,000,000,000 ÷ $500 = 20,000,000 tokens
```

### Traditional Banking Costs (Annual)
- FX spread (2.5%): $250,000,000
- Wire transfer & intermediary fees: $3,250,000
- Hedging costs (1%): $100,000,000
- Working Capital costs (0.5%): $50,000,000
- **Total: $403,250,000 annually**

### Onli Implementation Costs

**One-Time Setup:**
- Developer Access: $6,000
- Asset Management Fee ($10B AUM): $500,000
- Issuance Cost (20,000,000 tokens × $0.01): $200,000
- **Total Setup: $706,000**

**Ongoing Annual Costs:**
- Transaction fees: $0
- **Total Ongoing: $0**

### ROI Analysis
- Annual savings: $402,544,000
- Payback period: 0.64 days (15.4 hours!)
- Five-year cumulative savings: $2,012,014,000

## Full-Scale: $90.3 Billion

$90,300,000,000 ÷ $500 = 180,600,000 tokens

**Ongoing Annual Costs:**
- Transaction fees: $0

## Key Insights

1. **Much lower issuance costs** than previously calculated
2. **Payback period under 1 day** for both pilot and full-scale
3. **Onli becomes even more attractive** with correct asset-based pricing
4. **Higher asset denominations = fewer tokens = lower costs**
